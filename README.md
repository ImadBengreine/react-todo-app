# React Todo App

A clean, feature-rich todo application built with React and Tailwind CSS.

## Features

- ✅ Add, complete, delete todos
- ✏ Inline editing (double-click)
- 🔍 Filter: All / Active / Completed
- 💾 Persistent storage (localStorage)
- 🎯 Drag-and-drop reordering
- 📱 Fully responsive

## Live Demo

[View Live](https://ImadBengreine.github.io/react-todo-app)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- @dnd-kit (drag and drop)
- UUID (unique IDs)
